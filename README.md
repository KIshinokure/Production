## ?? Описание архитектуры системы
Проект реализует **цепочку задач (Pipeline)** в рамках мультиагентной системы автоматизации дискретного производства. Взаимодействие компонентов построено по асинхронной модели Pub/Sub с использованием брокера сообщений **NATS**.

1. **Оркестратор (Python):** Инициирует производственный заказ и выводит финальный аналитический отчет.
2. **Агент Диспетчеризации (Go):** Принимает первичный заказ и отправляет на проверку ресурсов.
3. **Агент Управления запасами (Go):** Резервирует сырье на складе.
4. **Агент Планирования загрузки (Go):** Выделяет оптимальную производственную линию.
5. **Агент Контроля качества (Go):** Проводит виртуальную симуляцию ОТК и закрывает заказ.
